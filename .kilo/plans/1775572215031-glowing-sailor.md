# Plan: Fix React Query Error - Query data cannot be undefined

## Problema
El hook `useAllSalesByBusinessId` usa una función de query que puede devolver `undefined`, lo cual viola las reglas de React Query.

## Causa Raíz
En `src/lib/api/sale.ts` línea 14:
```ts
return data.data;
```

Si el API retorna una respuesta que no tiene el formato esperado `{ data: [...] }`, o si hay algún caso donde `data.data` sea `undefined`, React Query lanza el error.

## Solución

Modificar `src/lib/api/sale.ts` para:
1. Retornar un array vacío como fallback en lugar de `undefined`
2. Manejar el caso donde `data` o `data.data` no existan

```ts
export async function getAllSalesByBusinessId({ businessId }: GetAllSalesByBusinessIdProps) {
    const { data } = await apiClient.get(
        salesRoutes.getAllSalesByBusinessId(businessId)
    );

    // Retornar array vacío si no hay datos
    if (!data) return [];
    return Array.isArray(data) ? data : data.data ?? [];
}
```
